# Brand Kit — YouTube Diagrams

## Status: ACTIVE

## Background
- Primary: `#FAFAF8` (off-white, like clean paper)
- Card/section fill: `#FFFFFF` (pure white cards on off-white bg)
- Code blocks: `#1E1E2E` (dark terminal background)

## Text
- Headers: `#1A1A1A` (near-black, bold hand-drawn feel)
- Body: `#3D3D3D` (dark gray for readability)
- Muted/labels: `#7A7A7A` (light gray for secondary info)
- Code text: `#A6E3A1` (green on dark terminal bg)

## Accent Colors
- Blue: `#6CB4EE` (soft blue — robot eyes, highlights, links)
- Orange: `#E8A848` (warm amber — arrows, callout borders, emphasis)
- Green: `#5CB85C` (checkmarks, success indicators)
- Red: `#E85D5D` (errors, X marks, warnings)
- Purple: `#B39DDB` (occasional accent for variety)
- Yellow: `#FFF3CD` (sticky note backgrounds)

## Strokes
- Section borders: `#1A1A1A`, 1.5-2px, slightly rounded corners (rx="12")
- Arrows: `#1A1A1A` or `#E8A848`, 2px, round linecap
- Connectors: `#1A1A1A`, 1.5px dashed for flow lines
- Hand-drawn feel: lines can have slight curve variation (use quadratic bezier)

## Typography
- Heading font: `Caveat` (weight 700, hand-drawn bold headers)
- Body font: `Patrick Hand` (clean hand-print, great readability)
- Code font: `monospace` (for terminal blocks)
- Heading sizes: 56-72px for main title, 36-42px for section headers
- Body sizes: 22-26px
- Labels/captions: 16-18px

## Layout Elements
- **Section boxes**: Rounded rectangles with thin black border, white fill, optional colored top bar (4-6px)
- **Sticky notes**: Slightly rotated rectangles with yellow (#FFF3CD) or accent fill, no border or light border
- **Terminal blocks**: Dark rounded rectangles with colored dots (red/yellow/green) in top-left, monospace text
- **Arrows**: Hand-drawn style with round caps, can be curved (quadratic bezier for organic feel)
- **Decorative sparkles**: Small star/sparkle shapes scattered sparingly (using `#E8A848` or `#6CB4EE`)

## Watermark/Credits
- Position: Bottom-right
- Font: Patrick Hand, 18-20px
- Color: `#CCCCCC` (very subtle)
- Format: `@ChannelName` (optional per diagram)

## Key Design Principles
1. Clean white space — don't overcrowd, let sections breathe
2. Bold headers catch the eye first, then content flows naturally
3. Muted accent palette — not rainbow, more like highlighted notebook
4. Terminal/code elements ground it in tech context
5. Hand-drawn feel should be consistent but not sloppy
6. Arrows and labels guide the viewer through the content flow
